## Tools
The following software development tools are used.
* Version control (Git)
* Project management (Maven)
* Cloud runtime (Heroku)

## Frameworks
The following frameworks are used.
* Java Servlets
* Spring Core technologies
* Spring Boot
* Spring Web MVC
* Thymeleaf
* Spring Data (Commons and JPA)
